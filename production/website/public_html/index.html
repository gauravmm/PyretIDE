What is Cutlass?

An IDE for Pyret

A brand new IDE designed for Pyret programmers by Pyret programmers.
Cutlass has features that makes functional programming easier.

Get Cutlass

Call Graph &
Code Analysis

Cutlass understands how control flows through your code.
Recursive functions, functions as arguments, functions called and calling functions — we keep track of all these.

Catches Common Errors

A fun without an end, a string without an ending quote or mismatched parenthesis, Cutlass catches these and more.

Indentation &
Highlighting

Messy code? The press of a single button cleans and re-indents your code for you.
Cutlass highlights the current word wherever it is used in the document.

Laskura — From Your Computer to Ours

Cutlass runs like any other offline IDE. Until you decide to code on the cloud.
Connect to a Laskura server and write code wherever you are.
With Laskura, Cutlass can store and run Pyret programs on a server. Designed for educators.
