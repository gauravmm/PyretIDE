read-sexpr

fun read-sexpr(str :: String) -> Any:

  

where:

  read-sexpr('((-13 +14 88.8) cats ++ "dogs")') is

    [[-13, 14, 444/5], 'cats', '++', ['string', 'dogs']]

end

Take a string as input, and parse it into an s-expression. Each s-expression is a number, symbol, string, or a list of s-expressions surrounded by parenthesis and separated by whitespace. Parenthesized lists are converted into Pyret lists, and strings are converted into a list ["string", <the-string>].

identical

fun identical(val1 :: Any, val2 :: Any) -> Boolean:  end

Checks if val1 and val2 were constructed from the same datatype. Raise an error on values that are not instances of datatypes constructed with data. List is a valid datatype to check identicality on.
